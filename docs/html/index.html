🌍 Umwelt-Datenlogger mit Arduino Mega 2560

📌 Projektüberblick

Dieses Projekt dient der kontinuierlichen Umweltüberwachung in Echtzeit mit einem Arduino Mega 2560. Es misst Temperatur, Luftfeuchtigkeit, Radioaktivität, Licht, Geräuschpegel, GPS-Daten und Gaskonzentrationen – und speichert diese zuverlässig im CSV-Format auf einer SD-Karte.

🔧 Verwendete Hardware

Detailed Description

GPS-Datenstruktur mit allen relevanten Informationen.

Diese Struktur enthält alle GPS-bezogenen Daten, die vom TinyGPSPlus Parser verarbeitet werden. Sie dient als standardisierte Schnittstelle für GPS-Informationen im gesamten System.
